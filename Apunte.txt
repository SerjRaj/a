Menú Puntos
--------o---------o-------o---------o-------
1. Click derecho en app -> New -> Android Resource Directory -> menu

2. Crear un Menu Resource File llamado 'mainmenu'

3. Ir a AndroidManifest.xml y buscar el activity de MainActivity y colocar la siguiente línea:

		android:theme="@style/Theme.AppCompat.Light.DarkActionBar"

4. En el MainActivity de Java, dejar solo:

			1. onCreateOptionsMenu
			2. onOptionsItemSelected

5. Dentro del mainmenu.xml declarar los items con:

		<item
			android:id="@+id/[Nombrex]"
			android:title="[Nombrex]"
			android:icon="@drawable/EscojeItem"
			app:showAsAction="always"/>

6. Dentro del onCreateOptionsMenu colocar:

			getMenuInflater().inflate(R.menu.mainmenu, menu);
			return true;

7. En el onOptionsItemSelected colocar:

			if(item.getItemId() == R.id.[nombreitem]){
			
			}

Fragmentos Dinámicos

--------o---------o-------o---------o-------


1. En el activitymain.xml colocar un FragmentContainerView, que debería verse así:

		<androidx.fragment.app.FragmentContainerView
    			xmlns:android="http://schemas.android.com/apk/res/android"
    			xmlns:app="http://schemas.android.com/apk/res-auto"
    			android:id="@+id/nav_host_fragment"
    			android:name="androidx.navigation.fragment.NavHostFragment"
    			android:layout_width="match_parent"
    			android:layout_height="match_parent"
    			app:defaultNavHost="true"
    			app:navGraph="@navigation/navegador" />

2. Click derecho a app -> New -> Android Resource Directory -> navigation -> y después declarar el Android Resource File para colocar
el nombre de 'navegador'

3. En el navegador, ubicarse en la Paleta y darle Add Destination al Inicio. Debe verse así:

 <fragment
        android:id="@+id/inicio"
        android:name="com.example.navegadordtc.Inicio"
        android:label="fragment_inicio"
        tools:layout="@layout/fragment_inicio">
        <action
            android:id="@+id/action_inicio_to_doble"
            app:destination="@+id/doble" />
    </fragment>
